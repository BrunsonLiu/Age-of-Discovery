## 1. 产品概述

大航海时代交互网站是一个以全屏古风航海地图为核心视觉的沉浸式交互体验平台，融合加勒比海盗主题，让用户通过游戏化交互体验15-17世纪大航海时代的探索、贸易与冒险。

- 目标用户：对大航海时代历史感兴趣的探索者、加勒比海盗文化爱好者、历史游戏玩家
- 核心价值：通过地图交互+事件驱动的叙事方式，让用户身临其境地感受大航海时代的壮阔与危险

## 2. 核心功能

### 2.1 用户角色
| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 舰队指挥官 | 直接进入 | 操控舰队、贸易、接受任务、探索航线 |
| 观察者 | 直接进入 | 浏览地图、港口信息、历史事件（只读模式） |

### 2.2 功能模块
1. **主地图页面**：全屏航海地图、港口标记、航线绘制、船只动画、风向洋流叠加
2. **舰队管理面板**：船只属性、船员状态、补给管理、装备升级
3. **港口交互面板**：商品买卖、补给修理、招募船员、接受任务
4. **事件决策面板**：随机事件触发、决策分支选择、结果反馈
5. **任务日志面板**：主线剧情推进、支线任务追踪、历史注释展示

## 3. 核心流程

用户进入网站后，以舰队指挥官身份开始大航海冒险。核心流程为：选择出发港口 → 准备物资 → 规划航线 → 航行推进 → 遭遇事件 → 做出决策 → 到达港口 → 贸易补给 → 接受新任务 → 继续航行。

```mermaid

    "进入网站" --> "选择角色模式"
    "选择角色模式" --> "指挥官模式"
    "选择角色模式" --> "观察者模式"
    "指挥官模式" --> "选择出发港口"
    "选择出发港口" --> "准备物资与船员"
    "准备物资与船员" --> "规划航线"
    "规划航线" --> "开始航行"
    "开始航行" --> "时间推进"
    "时间推进" --> "遭遇事件?"
    "遭遇事件?" --> "是: 事件决策"
    "遭遇事件?" --> "否: 继续航行"
    "事件决策" --> "结果结算"
    "结果结算" --> "到达港口?"
    "到达港口?" --> "是: 港口交互"
    "到达港口?" --> "否: 时间推进"
    "港口交互" --> "贸易/补给/任务"
    "贸易/补给/任务" --> "规划航线"
    "继续航行" --> "时间推进"
    "观察者模式" --> "浏览地图与历史"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝海洋色（#1a2a3a）+ 古铜金（#c9a96e）作为强调色
- 辅助色：羊皮纸米色（#f4e4c1）、深红警告色（#8b2500）、墨绿安全色（#2d5a27）
- 按钮风格：仿古铜扣圆角按钮，带金属质感边框和微妙阴影
- 字体：标题使用 Pirata One（海盗风格展示字体），正文使用 Crimson Text（古典衬线体）
- 布局风格：全屏地图居中，左右侧面板可折叠，底部固定操作栏
- 图标风格：航海主题手绘风格图标（罗盘、船锚、骷髅旗、酒杯等）
- 质感：羊皮纸纹理背景、皮革边框、墨水晕染效果

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主地图页面 | 全屏地图视图 | 深色古风海图底图，金色边框港口标记，虚线航线弧 |
| 主地图页面 | 港口标记层 | 金色锚点图标，hover发光效果，点击弹出羊皮纸风格弹窗 |
| 主地图页面 | 船只动画层 | 帆船图标沿航线移动，带尾迹粒子效果 |
| 主地图页面 | 环境叠加层 | 半透明风向箭头、蓝色洋流线条、红色海盗区域遮罩 |
| 舰队管理面板 | 侧边面板 | 左侧可折叠面板，羊皮纸背景，船只卡片竖向排列 |
| 港口交互面板 | 侧边面板 | 右侧可折叠面板，商品列表带价格涨跌指示 |
| 事件决策面板 | 居中弹窗 | 羊皮纸风格大卡片，顶部事件标题，中部叙述文字，底部决策按钮 |
| 任务日志面板 | 侧边面板 | 左侧可折叠面板，时间轴+任务卡片布局 |
| 底部操作栏 | 固定底栏 | 深色半透明背景，罗盘图标+推进按钮+状态提示 |

### 4.3 响应式设计
- 桌面优先设计，大屏全功能展示
- 平板：侧边面板改为抽屉式，底部操作栏精简
- 手机：地图全屏，面板改为底部抽屉，操作按钮简化

### 4.4 动效设计
- 页面加载：地图从模糊到清晰的"海雾散去"效果
- 船只航行：沿航线的平滑移动+轻微摇摆动画
- 事件触发：屏幕边缘红色闪烁+雷声震动效果
- 面板展开：从侧边滑入+羊皮纸展开动画
- 港口到达：港口标记放大脉冲+欢迎文字渐入
